加影“翡翠大酒家”（一）

Tag ：马来西亚 / 雪兰莪 / Kajang / 中餐
锚链：（一）, （二）

加影 “翡翠大酒家”

～龙虾象拔三文鱼刺身 - 三文鱼虽然较为普通但是非常新鲜。象拔蚌也是现宰现吃；因为象拔蚌属于贝类的一种，稍有一点点不够新鲜便马上会有一股腥味。龙虾的鲜度更是无可挑剔，因为就算肉已被取出切片，龙虾的脚还会偶尔划动一两下。至于刀功方面，厨师把象拔蚌的肉薄切，恰当的展现出它应有的脆度，完全不会韧。龙虾肉的筋都已被挑断，因此每一口的口感都很棒。另外附有一锅上汤；如果不喜欢生吃，把象拔或龙虾肉稍微灼一下，吃起来也另有一番风味。

～清蒸白苏丹鱼 - 属于顶级的野生淡水鱼之一，其美味早已远远抛离海鱼类。这里清蒸的酱汁，咸甜适中，却不会先兵夺主的抢了鱼的鲜味。尤其将混合了鱼的脂肪的蒸鱼酱汁，淋一点在白饭上，再配上一点的葱花，酱汁的优劣立判，无所遁形。

～猪肚汤 - 这锅汤绝对比一般的猪肚汤高班。这锅汤用了一只老母鸡，另加数量不少的鸡脚去熬煮高汤，过后才加入猪肚和胡椒粒长时间熬煮；可想而知它的味道是如何的鲜美。

～焖猪脚筋 - 用古法来焖制猪脚筋。火候掌握得很棒，猪脚筋软硬适中兼非常入味；是一道非常花功夫的一道佳肴。

～梅菜酿豆腐 - 肉馅稍微硬了一点，梅菜的味道也不太足够，有改进的空间。不过，他家自制的豆腐却非常柔软滑顺，豆香浓郁，可圈可点。

加影“翡翠大酒家”（二）

Tag ：马来西亚 / 雪兰莪 / Kajang / 中餐
锚链：（一）, （二）

加影 “翡翠大酒家”

-明火乳猪～虽然说不上特别好吃，不过算够得上水准之作。乳猪的肉软皮香，蘸酱恰到好处甜而不腻，再吃一两朵葱花，其微微的辛辣正好把胃口打开。

-清蒸多宝鱼～又叫 “比目鱼”，以细嫩的肉质著称。游水的多宝鱼口感虽然比不上野生的，但是胜在新鲜，肉质甜美。尤其是那块鱼肝，更是肥厚饱满，油脂丰富。

-红烧“鸭掌” ～ “鸭掌” 经长时间焖煮，口感软嫩，肉脂丰腴，胶质饱满。配上品质极好的鲍片，增添了口感的层次和风味；吃完后，两片嘴唇会黏住，可见胶质何其丰富。这道菜便是这次试菜的主角，应该会让兄弟们满意。

-家乡茄子～最后来的这道素菜凸显了主厨 Kent哥的功力。将茄子切段，快速灼水烫熟，然后淋上调过味道的酱油，最后撒上油葱、葱花和鸡肉丝。味道清新，口中的油腻感马上消失，让人感觉非常舒服。
